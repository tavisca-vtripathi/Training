Templar v1.7 - Development Setup Guide

Templar v1.7 is very easy to setup for development, provided you have the prerequisites installed

  1. Microsoft Visual Studio 2012 professional or higher
  2. SQL server 2008 R2 or higher
  3. .Net framework 4.5

Steps to follow:

  • Download Templar v1.7 setup from the download link provided to you
  • Unzip and open “Tavisca.Templar.Development.sln” in VS 2012
  • Run it and follow the installation steps as shown below

  1. Enter database server details for setup. Please ensure the DB user provided has permission to create a database.

  2. Enter new Templar database name and details for SuperUser account. Check the “Add Started kit” for creating a sample site.

  3. Please wait for setup to complete.

  4. Post successful setup you should see the following screen.

  5. Now login with your new credentials.

  6. If everything went well you should see the default site listing page as shown below.

  7. To view the site click on Current link. It should open a page like below
